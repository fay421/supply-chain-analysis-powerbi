# 📊 Supply Chain Delivery & Profit Analysis (Power BI)

## 🎯 Project Overview
This project analyzes a global supply chain dataset to identify key profit drivers and uncover the main causes of late deliveries.

The analysis focuses on two core business questions:
- 💰 Where is the business making the most profit?
- 🚚 How can we reduce late deliveries?

---

## 🛠️ Tools Used
- Power BI  
- DAX  
- Data Visualization  

---

## 📈 Key KPIs
- Total Sales: 6,640bn  
- Total Orders: 12bn  
- Total Profit: 716bn  
- Late Delivery Rate: 54.8%  

---

## 🔍 Key Insights

### 💰 Profit Drivers
- Central America and Western Europe generate the highest profit  
- Cleats and Footwear are top-performing categories  
- Standard Class drives the majority of revenue  

### 🚚 Delivery Performance
- Standard Class has the highest delay rate (~60%)  
- Same Day performs best (~5% delay)  
- Western Europe shows the highest regional delays  

---

## 💡 Key Business Insight
There is a clear trade-off between cost efficiency and delivery performance.

Lower-cost shipping (Standard Class):
- ✅ Drives higher profit  
- ❌ Causes higher delays  

---

## 🚀 Recommendation
Optimize Standard Class logistics and address regional inefficiencies to improve both profitability and delivery performance.
